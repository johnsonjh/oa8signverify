# `oa8sign` / `oa8verify`

## OpenSSL-based RSA-16384 + SHA3-512 + ASCII85 package signing tool

---

## Requirements

- [OpenSSL >= 1.1.1](https://www.openssl.org/)
- [Ruby >= 2.4.0](https://www.ruby-lang.org/)
- [ascii85gem >= 1.1.0](https://github.com/DataWraith/ascii85gem/)
